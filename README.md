# VIM Dance Studio

**Virtues In Motion Dance Studio** — Yorkville, IL

Premium redesign built on the Stitch "Ethereal Stage" design system.

## Stack
- Static HTML/CSS (single file, no framework)
- Served via `serve` on Railway
- Neon PostgreSQL (ready for backend integration)

## Local Dev
```bash
npm install
npm start
# → http://localhost:3000
```

## Deploy
Push to `main` → Railway auto-deploys.
